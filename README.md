KALINGA: Tiktok-like App for Pet Adoption<br>

This app uses Nextjs as frontend and backend and Supabase for the database.<br>
This is for a school project in Introduction to Engineering Design.<br>

type "npm run dev" to run<br>
/app/api - contains routes (this is like a middle man where the front will ask this route to talk to backend)<br>
/app/site - basically the front end<br>

/components - this where you put all stuff that can reusable<br>
/lib - the backend yey<br>
/public - put all images here, use .svg<br>
