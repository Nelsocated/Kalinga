KALINGA: Tiktok-like App for Pet Adoption

This app uses Nextjs as frontend and backend and Supabase for the database.
This is for a school project in Introduction to Engineering Design.

type "npm run dev" to run
/app/api - contains routes (this is like a middle man where the front will ask this route to talk to backend)
/app/site - basically the front end

/components - this where you put all stuff that can reusable
/lib - the backend yey
/public - put all images here, use .svg
